# Function of Exception Handling

> Exception handling adalah mekanisme dalam pemrograman yang digunakan untuk menangani kesalahan (error) yang terjadi selama eksekusi program. Fungsinya adalah untuk memastikan bahwa program dapat menangani kondisi tak terduga atau kesalahan tanpa menghentikan eksekusi secara tiba-tiba. Ini membantu menjaga kestabilan dan keandalan program.

### beberapa fungsi utama dari exception handling:

>Menangkap Kesalahan:

Exception handling memungkinkan program untuk "menangkap" kesalahan yang terjadi. Ketika sebuah kesalahan terjadi, program dapat berpindah ke kode yang khusus menangani kesalahan tersebut.

> Mencegah Program Berhenti Secara Mendadak:

Tanpa exception handling, kesalahan dalam program dapat menyebabkan program berhenti secara mendadak. Dengan exception handling, program dapat terus berjalan meskipun terjadi kesalahan, dengan cara menangani atau mengatasi kesalahan tersebut.

> Memberikan Informasi Kesalahan:

Exception handling memungkinkan pengembang untuk memberikan informasi yang jelas tentang kesalahan yang terjadi, sehingga lebih mudah untuk didiagnosis dan diperbaiki.

> Mengisolasi Kesalahan:

Kesalahan yang terjadi di satu bagian program dapat diisolasi sehingga tidak mempengaruhi bagian lain dari program. Ini memungkinkan bagian lain dari program tetap berjalan dengan normal.

> Meningkatkan Keamanan dan Kestabilan:

Dengan menangani kesalahan secara sistematis, program menjadi lebih aman dan stabil. Pengguna tidak akan mengalami crash yang tidak terduga atau kehilangan data.
